## 9. ตัวกรองในทะเบียนรายชื่อ

ในทะเบียนรายชื่อสามารถกรองเพิ่มเติมได้:

- ค้นตามที่อยู่ / บ้านเลขที่
- กรองตาม อสม. / ผู้บันทึก
- ล้างตัวกรอง

เมื่อเลือกหมู่บ้านจากตัวกรองหลัก ช่อง `กรองตาม อสม. / ผู้บันทึก` จะแสดงเฉพาะ อสม. หรือผู้บันทึกของหมู่นั้นเท่านั้น

## 10. การดูข้อมูลรายบุคคลและบันทึกน้ำหนัก/ส่วนสูง

หลัง login ในทะเบียนรายชื่อ แต่ละแถวจะมีปุ่ม `ดู/บันทึก`

เมื่อกดปุ่ม ระบบจะแสดงหน้าข้อมูลรายบุคคล ประกอบด้วย:

- ความดันโลหิต
- น้ำตาลในเลือด
- BMI
- CVD Risk
- กราฟแนวโน้มความดัน
- กราฟแนวโน้มน้ำตาล / BMI
- ตารางประวัติการคัดกรองและบันทึกสุขภาพ

การบันทึกน้ำหนัก/ส่วนสูง:

1. กด `ดู/บันทึก` ที่รายชื่อ
2. กรอกวันที่
3. กรอกน้ำหนัก กิโลกรัม
4. กรอกส่วนสูง เซนติเมตร
5. กด `บันทึก`

ข้อมูลน้ำหนัก/ส่วนสูงจะถูกเก็บไว้ในเครื่องนี้ด้วย `localStorage`

## 12. ข้อควรระวังด้านข้อมูลส่วนบุคคล

ระบบนี้มีข้อมูลรายชื่อประชาชน จึงควรใช้งานในเครื่องหรือเครือข่ายที่ปลอดภัย

ข้อควรระวัง:

- ไม่ควรส่งต่อไฟล์เว็บแอพพร้อมข้อมูลให้บุคคลที่ไม่เกี่ยวข้อง
- ไม่ควรเผยแพร่ผ่าน internet โดยไม่มีระบบ login ฝั่ง server
- ระบบ login ในเว็บแอพนี้เป็นการป้องกันเบื้องต้นในหน้าเว็บ static
- หากต้องใช้งานจริงหลายเครื่อง ควรพัฒนา backend login และฐานข้อมูลกลาง

## 13. ไฟล์สำคัญในระบบ
